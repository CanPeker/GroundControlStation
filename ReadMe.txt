That system which is called a ground control station has been created to communicate with a UAV which has been designed for competition thanks to a nanobeam-bullet system. 
GCS can manage UAV's axis movements by having Logitech joysticks. 
It has reciprocal data transferring such as real-time video streaming and flight data. All system has been written using Python programming language.
